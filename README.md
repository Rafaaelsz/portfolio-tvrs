# 🚀 Portfolio

Portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e experiências como Desenvolvedor Full Stack.

## ✨ Sobre

Este projeto foi criado com foco em performance, design moderno e experiência do usuário, reunindo alguns dos principais projetos que desenvolvi nas áreas de desenvolvimento web, automação, APIs e sistemas empresariais.

## 🛠️ Tecnologias

* Next.js
* React
* TypeScript
* Tailwind CSS

## 📂 Projetos em Destaque

### 📦 Orbit ERP

Sistema web completo para gestão de estoque e processos empresariais.

### 📈 Price Tracking API

API para monitoramento de preços e envio de alertas utilizando processamento assíncrono.

### 🎮 CS2 Box Analyzer

Extensão para Chrome/Edge que analisa risco, ROI e valor esperado em sites de abertura de caixas de Counter-Strike 2.

### 🔄 ConverteAi

Aplicação desktop desenvolvida em Python para conversão de diferentes formatos de arquivos.

## 🚀 Executando o projeto

```bash
npm install
npm run dev
```

## 🌐 Deploy

O projeto pode ser publicado facilmente utilizando plataformas como Vercel.
